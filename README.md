# Portfolio — Rita Knobel Borges

Site pessoal hospedado via GitHub Pages.

## Estrutura de pastas

```
rita-site/
├── index.html              ← Homepage
├── css/
│   ├── base.css            ← Estilos globais (header, footer, variáveis)
│   └── home.css            ← Estilos exclusivos da homepage
├── js/
│   └── main.js             ← Scripts (menu mobile, etc.)
├── images/
│   └── perfil.png          ← ⚠️ Adicione aqui a sua foto!
└── paginas/
    ├── identidades-visuais.html
    ├── meus-jogos.html
    ├── minha-arte.html
    └── sobre-mim.html
```

## Como usar

1. Faça upload de `perfil.png` na pasta `images/`
2. Faça o push para o branch `main` (ou `gh-pages`)
3. Ative o GitHub Pages nas configurações do repositório
